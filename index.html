☁️
設定情報のクラウド同期
テーマ・温度単位・最終検索都市・ニュースカテゴリの選択状態がサーバーに保存されます。複数の端末やブラウザからアクセスした際も、同一の設定が自動的に復元されます。
🏙️
最終閲覧都市の自動復元
前回ご利用時に検索された都市の天気情報が、次回ログイン時に自動で表示されます。毎回の入力操作を省略することが可能です。
📋
利用履歴の記録
天気検索およびニュースカテゴリの閲覧履歴が日時情報とともにデータベースに記録されます。お客様自身の利用状況を管理画面からご確認いただけます。
🔒
データの完全分離・セキュリティ保護
各ユーザーのデータはRow Level Security（行レベルセキュリティ）により厳密に分離されています。他のユーザーがお客様の設定・履歴にアクセスすることは一切できません。
🗄️
データ管理基盤について
本アプリケーションのユーザー情報・設定データ・利用ログは、Supabase（PostgreSQLベースのクラウドデータベース）にて一元管理されております。認証済みユーザーのアカウント情報・登録日時・最終ログイン日時は Supabase の Authentication 管理画面にて確認可能であり、設定データおよび行動ログは user_settings / user_logs テーブルにてユーザー ID 単位で管理されています。

🔒 プライバシーポリシー

Luna & Elma は、快適な利用体験を提供するために以下のデータを端末内またはクラウドに保存します。

  • 位置情報 — 現在地の天気取得時のみ使用。外部に送信後、座標は端末に保持しません。
  • 設定・履歴 — テーマ、単位、検索履歴、お気に入りは localStorage に保存されます。
  • ニュース既読・ブックマーク — localStorage に保存。端末外には送信されません。
  • 要約キャッシュ — AI が生成した要約テキストを最大7日間 localStorage に保存します。
  • アカウント情報 — ログイン時は Supabase（Northeast Asia / Tokyo リージョン）にメールアドレスと設定が保存されます。
  • チャット履歴 — ブラウザの localStorage のみに保存。サーバーには送信されません。

第三者への個人情報販売は行いません。詳細はお問い合わせください。
